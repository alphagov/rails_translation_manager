# RailsTranslationManager

Tasks to manage translation files

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rails_translation_manager'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails_translation_manager
